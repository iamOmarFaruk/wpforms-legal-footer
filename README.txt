== Description ==

**WPForms Legal Footer** allows you to add a customizable text link directly below the submit button of any WPForms form. This is perfect for GDPR compliance, Terms of Service, or Privacy Policy links.

**Features:**
*   **Official Integration**: Uses the WPForms "Settings" tab for a native feel.
*   **Customizable**: Change text, URL, color, font size, and alignment.
*   **Safe**: Won't break your form layout.
*   **Lightweight**: Minimal code, high performance.

== Installation ==

1.  Upload the plugin folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **WPForms > Edit Form > Settings**.
4.  Find the **"Legal Footer"** tab and configure your link!
